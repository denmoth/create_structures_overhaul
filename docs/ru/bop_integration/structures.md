# Biomes O’ Plenty Integration

Covers structures that appear when **Biomes O’ Plenty** is installed.

## Example Structures
- Jungle Workshop (Rainforest)
- Frozen Mill (Snowy biomes)
- Amber Ruins (Ominous Woods)

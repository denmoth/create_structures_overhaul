Featured addons

Popular community projects that build on Structures Overhaul. Install them alongside the core mod for expanded exploration.

BOP Integration

Adds structure sets for Biomes O' Plenty, including new palette swaps for redwood forests and lavender fields.

  • Requires Biomes O' Plenty 1.20.1-17.2+
  • Includes datapack for custom loot and music discs
  • Maintained by @AuroraForge

Engineer's Manual

Interactive Patchouli guide that explains each structure, contraption, and recommended progression.

  • Ships with questbook integration for FTB Quests
  • Supports multiplayer syncing
  • Maintained by @Redstoned

Nether Expedition Pack

Experimental expansion that ports Create machinery into the Nether with basalt citadels and blaze-powered elevators.

  • Requires Structures Overhaul 1.5+
  • Currently in beta â€” expect schematic changes
  • Maintained by @Denmoth

Submit your addon

Share your addon to get featured in future releases.

  1. Publish your repository.

    Use a permissive license and include install instructions plus datapack requirements.

  2. Create a pull request.

    Add your addon to the addons.json index on GitHub with name, author, version, and download links.

  3. Showcase it to the community.

    Post screenshots or video tours in the Discord #showcase channel for feedback.

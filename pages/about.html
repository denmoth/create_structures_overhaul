About

Learn more about the project.

Project vision

Create: Structures Overhaul is a world generation addon for the Create mod. It enriches exploration by adding thematic structures to your world. It is open source and welcomes contributions.

The addon focuses on replayability, exploration, and teaching Create mechanics through environmental storytelling. Every structure contains thoughtful contraptions designed to be dismantled, studied, or rebuilt elsewhere.

Design pillars

  • Automation ready: Each structure includes plug-and-play contraptions that demonstrate Create mechanics.
  • Adventure friendly: Exploration rewards lore, schematics, and loot that support progression.
  • Pack builder tools: Datapacks and API hooks make it easy to integrate with modpacks and worldgen mods.

Roadmap

Upcoming features

  • Narrative questline that unlocks structure upgrades.
  • Nether and End expansion packs.
  • Exportable contraption schematics for sharing builds.

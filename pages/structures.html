Structure families

Structures are organised into thematic families. Each family shares palette rules, loot pools, and contraption tutorials.

Industrial complexes

Large multi-room facilities showcasing Create automation loops. Expect drills, sequenced assembly, and fluid processing.

  • Ashen Foundry
  • Clockwork Refinery
  • Verdant Assembly Line

Frontier outposts

Smaller self-contained bases that introduce players to transportation mechanics and supply chains.

  • Aurora Observatory
  • Workshop Caravan
  • Skyport Docks

World curiosities

Rare generation pieces used for lore breadcrumbs, unique loot, and advancement triggers.

  • Temporal Archive
  • Vault of Echoes
  • Gilded Relay

Loot profiles

Each structure family uses layered loot tables. The primary roll grants Create components while secondary rolls include datapack hooks for modpack authors.

Default loot layout

  • Core crate: shafts, cogwheels, mechanical pistons, andesite alloy.
  • Specialty barrel: deployer hands, redstone contacts, schematicannon ammo.
  • Data cache: lore books that trigger advancements or guide quests.
  • Bonus roll: integration tokens consumed by addon datapacks.
Override loot pools by copying data/cso/loot_tables/ into a datapack and editing the JSON entries.

Configuration

The mod exposes multiple layers for customizing world generation.

Global config

Edit config/create-structures-overhaul.toml to toggle structure families, adjust spacing, or switch loot difficulty presets.

Datapack overrides

Use data/<pack>/worldgen/structure_set to redefine spawn frequencies and add modded biome tags.

Per-structure JSON

Fine-tune block palette swaps, contraption templates, and jigsaw rules inside data/cso/structures.

Remember to run /reload after editing datapacks. Use /cso debug palette to print active palette rules.
